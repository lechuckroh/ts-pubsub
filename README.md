# Pub/Sub in TypeScript

Pub/Sub Deno library.

## Requirements

- Deno
- [task](https://taskfile.dev)

## Run

```bash
# lint
$ task lint

# formatting
$ task fmt
```

## Test

```bash
# Run test
$ task test

# test coverage (console output)
$ task cover-pretty

# test coverage (file output)
$ task cover-lcov

# remove test coverage output file
$ task rm-cov
```
